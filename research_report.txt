## 1. Executive Summary

This research project presents a comprehensive shipping efficiency analysis dashboard for Nassau Candy Distributor, a fictional candy manufacturing company inspired by Roald Dahl's Charlie and the Chocolate Factory. The project leverages data analytics and interactive visualization techniques to analyze shipping performance across multiple factory locations and customer regions.

The dashboard provides actionable insights into route efficiency, geographic bottlenecks, shipping mode performance, and operational optimization opportunities. Key findings include identification of the most efficient shipping routes, regional performance variations, and recommendations for cost-time tradeoffs in shipping operations.

The technical implementation utilizes Python's data science ecosystem (pandas, numpy) with Streamlit for web deployment and Plotly for interactive visualizations, ensuring accessibility and real-time analytical capabilities.

## 2. Introduction

### 2.1 Project Background

Nassau Candy Distributor operates five specialized factories producing various confectionery products:
- **Lot's O' Nuts**: Chocolate products (Utah, USA)
- **Wicked Choccy's**: Premium chocolate bars (Florida, USA)
- **Sugar Shack**: Sugar-based candies (Texas, USA)
- **Secret Factory**: Specialty items (Illinois, USA)
- **The Other Factory**: Unique products (Tennessee, USA)

The company serves customers across multiple US regions and states, utilizing various shipping modes to deliver products efficiently.

### 2.2 Research Objectives

The primary objectives of this research project are:

1. **Route Efficiency Analysis**: Identify the most and least efficient shipping routes from factories to customer regions
2. **Geographic Performance Mapping**: Create visual representations of shipping performance across US states
3. **Shipping Mode Optimization**: Compare performance across different shipping methods
4. **Bottleneck Identification**: Pinpoint geographic and operational bottlenecks
5. **Interactive Dashboard Development**: Create a user-friendly interface for real-time analysis

### 2.3 Significance of the Study

This project demonstrates the application of data analytics in supply chain optimization, providing:
- Cost reduction opportunities through route optimization
- Improved customer satisfaction through faster delivery times
- Data-driven decision making for shipping strategy
- Interactive visualization techniques for stakeholder communication

## 3. Data Description and Sources

### 3.1 Dataset Overview

The primary dataset contains shipping order information with the following key attributes:

- **Order Information**: Order ID, Order Date, Ship Date
- **Product Details**: Product Name, Division (Chocolate, Sugar, Other)
- **Geographic Data**: Customer Region, State/Province
- **Shipping Details**: Ship Mode, Lead Time
- **Factory Information**: Factory assignments for each product

### 3.2 Data Structure

**Main Dataset Columns:**
- Order ID (unique identifier)
- Order Date (datetime)
- Ship Date (datetime)
- Ship Mode (Standard Class, First Class, Second Class, Same Day)
- Customer ID
- Product Name
- Region (West, East, Central, South)
- State/Province (2-letter state codes)
- Lead Time (calculated: Ship Date - Order Date)

**Factory Data:**
- Factory Name
- Geographic coordinates (Latitude, Longitude)
- Product assignments

**Product-Factory Mapping:**
- Product Name
- Division
- Assigned Factory

### 3.3 Data Quality Assessment

Initial data quality checks revealed:
- No missing values in critical columns
- No duplicate records
- Consistent date formatting (with mixed day-first format)
- Valid geographic state codes
- Reasonable lead time distributions

## 4. Data Preprocessing and Cleaning

### 4.1 Date Standardization

```python
df['Order Date'] = pd.to_datetime(df['Order Date'], format='mixed', dayfirst=True)
df['Ship Date'] = pd.to_datetime(df['Ship Date'], format='mixed', dayfirst=True)
```

### 4.2 Lead Time Calculation

```python
df['Lead Time'] = (df['Ship Date'] - df['Order Date']).dt.days
df = df[df['Lead Time'] >= 0]  # Remove invalid negative lead times
```

### 4.3 Data Integration

## 5. Methodology and Analytical Framework

### 5.1 Analytical Approach

The research employed a multi-dimensional analytical framework:

1. **Route Analysis**: Factory-to-region and factory-to-state performance metrics
2. **Geographic Analysis**: State-level performance mapping
3. **Shipping Mode Analysis**: Comparative performance across delivery methods
4. **Bottleneck Identification**: Statistical outlier detection

### 5.2 Key Performance Indicators (KPIs)

- **Average Shipping Lead Time**: Mean days from order to delivery
- **Lead Time Variability**: Standard deviation of delivery times
- **Total Shipments**: Volume of orders per route/region
- **Route Efficiency Ranking**: Performance-based route prioritization

### 5.3 Statistical Methods

- **Descriptive Statistics**: Mean, standard deviation, min/max calculations
- **Grouping and Aggregation**: pandas groupby operations for route analysis
- **Ranking Algorithms**: Sort-based performance ranking
- **Congestion Analysis**: Multi-criteria filtering for bottleneck identification

## 6. Key Findings and Insights

### 6.1 Route Efficiency Analysis

**Top Performing Routes:**
1. Secret Factory to Central (Lowest average lead time)
2. Sugar Shack to South
3. Lot's O' Nuts to West

**Underperforming Routes:**
1. Wicked Choccy's to East (Highest average lead time)
2. The Other Factory to Central
3. Secret Factory to East

### 6.2 Geographic Performance Insights

**High-Performing States:**
- Texas (TX): Consistently low lead times
- California (CA): Efficient coastal shipping
- Florida (FL): Proximity to Wicked Choccy's factory

**Bottleneck States:**
- New York (NY): Highest average lead times
- Massachusetts (MA): Significant delays
- Pennsylvania (PA): Operational challenges

### 6.3 Shipping Mode Performance

**Performance Ranking:**
1. **Standard Class**: Best balance of speed and volume (lowest average lead time)
2. **Second Class**: Good performance for expedited needs
3. **First Class**: Mixed results, higher variability
4. **Same Day**: Counter-intuitive longer lead times despite premium pricing

### 6.4 Congestion-Prone Areas

**High Volume + High Lead Time Regions:**
- East Region: High shipment volume with above-average lead times
- Central Region: Significant operational bottlenecks
- Northeast States: Geographic and logistical challenges

### 6.5 Cost-Time Tradeoff Analysis

**Key Insights:**
- Standard Class provides optimal cost-efficiency ratio
- "Expedited" services don't always deliver faster service
- Regional proximity significantly impacts performance
- Factory specialization affects route efficiency
